🔥 URGENT: Firebase Firestore Setup Required!

You're getting permission errors because Firestore security rules aren't set up yet.

QUICK FIX (5 minutes):

Step 1: Go to Firebase Console
https://console.firebase.google.com/project/al-ameen-stable/firestore

Step 2: If you see "Create database" button, click it
   - Choose "Start in test mode" (for development)
   - Select your region (e.g., us-central1)
   - Click "Enable"

Step 3: Go to Rules tab
https://console.firebase.google.com/project/al-ameen-stable/firestore/rules

Step 4: Replace ALL rules with this (copy everything):

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Allow read/write for all during development
    // IMPORTANT: Change this before production!
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Step 5: Click "Publish"

✅ Your app will work immediately after this!

Note: This is for DEVELOPMENT ONLY. Before going live, use the secure rules from DEPLOYMENT.md

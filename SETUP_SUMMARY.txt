📋 NEXT STEPS TO DEPLOY:

STEP 1: Set Up Firebase (10 minutes)
───────────────────────────────────────
1. Go to https://console.firebase.google.com/
2. Create new project: "al-ameen-stable"
3. Enable Authentication:
   - Build > Authentication > Get Started
   - Enable "Email/Password"
4. Create Firestore Database:
   - Build > Firestore Database > Create
   - Choose "Production mode"
5. Get your config:
   - Settings ⚙️ > Project settings
   - Add web app
   - Copy firebaseConfig

STEP 2: Update Configuration
───────────────────────────────────────
1. Open: firebaseConfig.js
2. Replace placeholder values with your Firebase config
3. Save the file

STEP 3: Set Security Rules
───────────────────────────────────────
1. In Firebase Console: Firestore > Rules
2. Copy rules from DEPLOYMENT.md
3. Publish

STEP 4: Test Locally
───────────────────────────────────────
Run: npm start
Press: w (web), a (Android), or i (iOS)

Create your first admin account!

STEP 5: Deploy to Production
───────────────────────────────────────
For Web (FREE hosting):
  npm install -g firebase-tools
  firebase login
  firebase init
  npx expo export:web
  firebase deploy

For Mobile Apps:
  npm install -g eas-cli
  eas build --platform android
  eas build --platform ios

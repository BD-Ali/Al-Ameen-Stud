========================================================================

Each reminder is stored in Firebase with:

🎯 HOW IT WORKS:
========================================================================

1. **Creating a Reminder:**
   - Admin selects a horse and clicks "Add Reminder"
   - Fills in note, date, and time
   - System creates notification scheduled for that exact date/time
   - Saves to Firebase with notification ID
   - User gets confirmation message

2. **Notification Delivery:**
   - expo-notifications schedules the notification locally on device
   - At the specified date/time, notification fires
   - Shows up in notification tray
   - Plays sound and vibrates

3. **Deleting a Reminder:**
   - Admin deletes reminder
   - System cancels the scheduled notification
   - Removes from Firebase database
